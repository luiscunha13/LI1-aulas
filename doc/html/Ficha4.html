A função deslocaesq recebe uma lista e desloca cada elemento da lista, n posições para a esquerda.

A função é definida da seguinte forma:

deslocaesq :: Int -> [Int] -> [Int]
deslocaesq a [] = []
deslocaesq n l@(h:t) | n == 0 = (h:t)
                     |n < 0 = error "n inválido"
                     |otherwise = let chunk = mod n (length l) in drop chunk l ++ take chunk l

Exemplos de utilização:

>>> deslocaesq 2 [1,2,3,4]
[3,4,1,2]

>>> deslocaesq 3 [5,6,7,8,9]
[8,9,5,6,7]
